## 笔记说明 

笔记中基本涉及的代码基本都以Go语言为主，且附带JS的实现版本与对应说明。

当然由于Go语言本身的简洁性，本笔记无需预先学习Go语言相关语法，拥有基本的C系语言都能轻松阅读。
